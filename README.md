# SSU_OS
2020년 가을학기 운영체제 수업에서 수행했던 과제의 소스코드 입니다.  
모든 코드는 우분투 리눅스 18.04 LTS 환경에서 Vim 편집기를 사용하여 C언어로 작성했습니다.

성적은 전체 91명 중 1등으로 A+ 받았습니다.

## 설계과제

주어진 명세대로 프로그램을 구현하는 과제입니다.
